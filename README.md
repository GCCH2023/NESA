# NESA
一个C++开发的将NES文件反编译为C语言代码的项目。


## 构建运行项目

按照以下步骤构建运行项目:

1. 克隆仓库:
   ```bash
   git clone https://github.com/GCCH2023/NESA.git
   ```
2. 打开克隆下来的项目文件夹，双击里面的 NESA.sln 文件，使用 Visual Studio 2013 或更高版本的 Visual Studio 打开;

3. 使用 Visual Studio 编译运行项目。
